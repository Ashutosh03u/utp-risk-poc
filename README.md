# Satellite-Based UTP Risk Detection POC

## Overview

This project demonstrates how industrial activity signals derived from satellite observations can support early warning systems for UTP (Unlikely To Pay) credit risk detection.

## Features

- Industrial activity monitoring
- UTP risk scoring
- Risk classification
- Trend visualization
- Dashboard monitoring
